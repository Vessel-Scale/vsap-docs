# All Available Assessments

The **All Available Assessments** tab is the second tab in the Available Assessments area of your Client Portal. It shows every assessment type your organization offers as a card grid — so you can see what's available and start a new one at any time, without waiting for your Account Executive to assign it.

> This tab sits alongside **[Your Assessments](assessments-table.md)**, which lists the assessments already assigned to or created by your account.

---

## Browsing the card grid

Each card displays:

- **Icon** — a color-coded visual for quick recognition
- **Name** — the assessment type name

Click any card to open the **Assessment Detail** modal with more information.

![All Available Assessments card grid](../assets/screenshots/client-portal/cp-all-available-assessments.png)

---

## Assessment Detail modal

When you click a card, a modal opens showing:

- The assessment **icon and name**
- A **description** of what the assessment evaluates and who it's designed for

### For client users

You'll see a **Create This Assessment for My Account** button. Clicking it takes you directly to the Create Assessment form, pre-filled with your account and the selected assessment type. Fill in any remaining details and submit to create your draft.

![Assessment detail modal — client view](../assets/screenshots/client-portal/cp-assessment-detail-modal-client.png)

Once created, the assessment appears in your **[Your Assessments](assessments-table.md)** table with a **Draft** status. Use the **View / Edit** button on that row to open and manage it.

### For administrators and Account Executives (preview mode)

When viewing the portal in **[Preview Mode](index.md#previewing-as-an-administrator)** (via "View as Client"), you'll see an informational notice instead of the create button:

> *"This feature is only available to client users. In preview mode, use the Create button in the sidebar to create an assessment for this account."*

This is intentional — to create an assessment on behalf of a client, use the standard **Create Assessment** flow from the admin sidebar.

---

## What assessments appear here?

The grid shows all **active, top-level** assessment types in your organization's library — the same assessment definitions that appear in the Library. Sub-assessments (child assessments nested inside a parent) are not shown here.
