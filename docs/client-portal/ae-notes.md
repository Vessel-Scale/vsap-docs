# Account Executive Notes

This page explains what **Account Executives** and **Administrators** see in the Client Portal compared to what **clients** see, and how to manage client access.

---

## What you see vs. what clients see

### Library (NEW)

**Clients see:**
- A searchable, filterable library of all assessments they have permission to launch
- Ability to filter by group/category
- Assessment cards with descriptions and group labels
- Can launch any assessment directly

**You (AE/Admin) see:**
- Same Library view when using "View as Client"
- In the admin area: Ability to manage which assessments appear in the client library
- Control over assessment visibility per account/client

### Completed Surveys

**Clients see:**
- All assessments assigned to their account
- Current status (Not Started, In Progress, Completed)
- Ability to take/continue assessments
- Last updated date
- Copy link button to share with colleagues

**You (AE/Admin) see:**
- Same view when using "View as Client"
- In the admin area: Can assign new assessments to accounts
- Can view completion progress across all your accounts
- Can see which clients haven't started assigned assessments

### Industry Benchmarking

**Clients see:**
- How their organization's resilience score compares to industry average
- Visual gauge showing their position in the score range
- General industry context

**You (AE/Admin) see:**
- Same benchmarking view when using "View as Client"
- In the admin area: Can see benchmarking data across all accounts
- Can compare multiple accounts to industry averages
- Can use this data for strategic discussions with clients

---

## Key differences in admin areas

When you're NOT viewing as a client, you have additional tools:

- **Assign Assessments** — select which assessments appear in the library for each account
- **View Account Overview** — see all accounts you manage and their assessment status
- **Configure Dashboard** — customize which components appear in the client portal
- **View Analysis Reports** — access detailed results and analysis for client assessments
- **Manage Permissions** — control what each client user can see and do

---

## Previewing what clients see

To see exactly what a client sees:

1. Go to **Account Details** for the client
2. Click **View as Client** button
3. You'll enter preview mode with a yellow banner at the top
4. Browse the portal as if you were the client
5. Click **Return to Account** to go back to admin view

This is the best way to troubleshoot client issues or verify that clients can access what they need.

## Using Library with clients

### Why the Library matters

The new **Library** feature gives clients self-service access to assessments. Instead of waiting for you to assign assessments one-by-one, clients can:
- Browse available assessments
- Understand what each assessment evaluates
- Launch new assessments on their own schedule

### What to communicate to clients

When rolling out Library access to clients:

1. **Orientation** — show them the Library section and how to search/filter
2. **Explain groups** — let them know what each category/group represents
3. **Permissions** — clarify which assessments they have access to
4. **Support** — let them know how to get help if they have questions

### Managing library access

You control which assessments appear in the client's Library through:
- **Account Settings** — assign/remove assessments per account
- **Assessment Permissions** — control fine-grained access if needed
- **Groups/Categories** — organize assessments logically so clients can find what they need

## Common AE tasks

### Assigning a new assessment to a client

1. Go to **Account Details** for the client
2. Click **Assessments** or **Manage Assessments** tab
3. Click **Add Assessment**
4. Select the assessment from the list
5. Click **Assign**
6. The assessment appears in the client's **Library** immediately

### Checking client progress

1. Go to **Account Details** for the client
2. View the **Assessments** tab to see:
   - Which assessments have been started
   - Current completion status
   - Last update date
3. Use filters to focus on assessments that need attention (e.g., "Not Started" status)

### Understanding low completion rates

If a client has many "Not Started" assessments:
- Check if they understand how to use the Library
- Verify they have the right permissions
- Reach out to confirm the assessments are relevant to their business
- Offer a walkthrough if needed

### Sharing assessment links

Clients can copy assessment links from the portal, but you can also:
- Share links via email from your admin dashboard
- Create assignment batches for multiple users
- Track who's completed assessments via link
