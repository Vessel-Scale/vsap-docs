# Getting Started

The **Vessel Smart Assessment Platform (VSAP)** helps you manage the full lifecycle of client assessments — from onboarding accounts and designing question templates, to collecting responses, analyzing results, and publishing shareable web reports.

This quick-start guide walks you through the core workflow in seven steps. Each step links to a dedicated page with screenshots and details. For deeper reference on any feature, use the main navigation menu above.
