# Step 6 — Create a Web Report

A **Web Report** publishes your assessment findings as a branded, shareable web page that clients can view in their browser.

---

## Two-Part Process

Creating a web report involves two steps:

1. **Build the report template** in **Settings → Web Reports**
2. **Attach and publish the report** from the Assessment Details page

---

## Step 1 — Set Up a Web Report Template

Navigate to **Settings → Web Reports** to create a report template. Templates control the layout, branding, and sections shown in the published report.

[Full guide: Web Reports](../settings/web-reports.md){ .md-button }

## Step 2 — Attach to an Assessment

From the **Assessment Details** page, click the **Web** button in the top action bar.

Select a report template from the list and publish it. Once published, a shareable URL is generated that you can send directly to the client.
