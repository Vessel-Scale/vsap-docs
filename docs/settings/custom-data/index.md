# Custom Data

Manage organization-specific reference data used throughout the platform. Custom data includes industry classifications (NAICS codes), regional associations, and media assets that appear across assessments, reports, intake forms, and branding.

## Overview

The Custom Data section provides centralized management for three key data types:

- **[NAICS Codes](naics.md)** - Industry classification system that categorizes businesses by type and sector
- **[Regional Associations](rma.md)** - Regional Manufacturer Associations and local support organizations
- **[Media Library](media-library.md)** - Images and media assets for use in reports, emails, and content

## Where Custom Data is Used

Custom data appears throughout the platform:

- **[Intake Forms](../intake-forms.md)** - Users select NAICS codes and regional associations when creating accounts
- **[Web Reports](../web-reports.md)** - Media and reference data are used in report templates and content sections
- **[Email Templates](../email-templates.md)** - Logos and images from media library are included in emails
- **[Industries](../../industries/index.md)** - NAICS codes provide the foundation for ecosystem analysis
- **[Dashboard](../../dashboard/pivot-table.md)** - Custom data enables pivoting and grouping by classification and region
- **Account Details** - Display associated NAICS codes and RMA assignments
